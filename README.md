# Stock App

This is a simple Java application that fetches the current price of the Dow Jones Industrial Average (^DJI) using the Yahoo Finance API.

## Building and Running the Project

To build the project, run the following command:

```bash
./gradlew build
```

To run the application, use the following command:

```bash
./gradlew run
```
